KET 五个月备考全程计划

井说英语 · 永汲教育 Well Talk  |  5个月 · 三个阶段 · 全方位备考

01
第一阶段 · 1.5个月
🎯 语法奠基 · 话题融合
以真实话题为载体，系统掌握KET核心语法点。词汇、语法、表达不孤立学习，而是融入主题场景中理解吸收。
02
第二阶段 · 2个月
🎯 专项突破 · 因材施教
根据孩子具体情况，一个板块一个板块攻克或综合提升。口语+听力+写作+阅读+题型技巧全覆盖。
03
第三阶段 · 1.5个月
🎯 全真模考 · 查漏补缺
全方位题型练习。限时模考→错题分析→针对性补漏。适应考试节奏，建立考场心态。

第一阶段：语法奠基 · 话题融合（第1-6周）

📋 课前诊断

诊断项目内容
语法测试覆盖KET全部语法点的基础测试卷（45分钟），识别孩子的语法薄弱项
词汇量摸底从KET 1500词中抽取100个核心词进行听写+中英互译测试
口语评估5分钟自由对话，评估发音、流利度、句型复杂度
听力初测1篇短对话+1篇独白（KET难度），摸清听力起点
学习习惯评估与家长沟通孩子的学习习惯、专注力、每天可用时间
💡 诊断后产出： ① 孩子专属语法弱点清单 ② 词汇掌握热力图 ③ 个性化学习节奏建议

📚 核心教学理念

不孤立教语法。每个语法点都嵌入一个真实的话题场景中学习。比如，在「我的日常生活」话题中学习一般现在时和频率副词；在「上周末发生的事」中学习一般过去时；在「去超市买东西」中学习可数/不可数名词和数量表达。

📅 1-6周教学内容

周次话题语法点词汇量表达训练
第1周 🏠 我的家庭 · 自我介绍 一般现在时 be动词 人称代词 40词 介绍家庭成员、描述关系
第2周 🌅 日常生活 · 时间表达 频率副词 时间介词 一般现在时(三单) 40词 描述一天的活动安排
第3周 🏫 我的学校 · 课程与活动 there be句型 some/any 方位介词 40词 描述教室和学校设施
第4周 📅 上周末 · 过去的经历 一般过去时 不规则动词 was/were 40词 讲述周末做了什么
第5周 👗 购物与衣服 · 比较选择 形容词比较级 最高级 too/enough 40词 比较商品、表达偏好
第6周 🌤️ 天气与计划 · 展望将来 一般将来时 be going to will/won't 40词 做周末计划、天气预报

⏱️ 课时节奏

时间内容
第1节25分钟新知识点引入 + 核心语法/词汇教学 + 小练习
休5-10分钟休息
第2节25分钟强化练习 + 口语输出 + 纠错反馈 + 作业布置

每天50分钟高效学习。25分钟短课时保持注意力集中，两节连上形成"输入→输出"闭环。

📅 第一阶段的6周（共60节课）

每周5天上课，每天2节。每周围绕1个话题，12节课完成一个完整闭环。

天第1节（25分钟）第2节（25分钟）说明
第1天 🆕 话题词汇引入（10词）+ 场景短文朗读 📖 短文理解 + 词汇游戏巩固 建立话题认知，输入为主
第2天 🔤 语法点讲解（25分钟深度讲透1个语法点） ✍️ 语法练习 + 即时纠错 核心语法，当天消化
第3天 📖 话题词汇（+10词）+ 语法结合练习 🗣️ 情境对话 + 口语输出 词汇扩展+开口表达
第4天 📝 语法进阶练习 + 话题阅读 ✏️ 短篇写作（结合话题和语法） 读写结合应用
第5天 🔄 周复习：词汇听写 + 语法回顾 📊 周测验 + 口语评估 + 作业布置 检验+反馈+下周预告
🎯 25分钟课的核心原则：每节课只有一个焦点。不贪多。语法点拆成最小单元，一个25分钟只讲一个点。词汇每次只学10个新词，保证当场记住。
📌 第一阶段重点：不追求速度，确保每个语法点扎实掌握。语法错误当场纠正，建立正确的语法直觉。词汇通过话题自然习得，不死记硬背。

第二阶段：专项突破 · 因材施教（第7-14周）

🔍 第二阶段前的评估

根据第一阶段的学习表现和阶段测试成绩，判断孩子的强弱项分布，决定第二阶段采用「逐一攻克模式」还是「综合提升模式」。

判断标准适用模式说明
某一两个板块明显落后（如听力得分<50%，阅读得分>80%） 逐一攻克 集中力量补短板，一块一块打穿。优先攻克最弱的板块。
各板块水平相对均衡（差距<20%） 综合提升 每周围绕一个话题，听说读写综合训练，整体拉升。

🔤 模式A：逐一攻克（推荐大多数孩子）

阶段板块周期核心内容
2.1 🎧 听力突破 2-3周 5个Part逐一攻克：图片选择(Part1)→填空(Part2)→匹配(Part3)→对话选择(Part4)→长填空(Part5)。精听三遍法：第1遍做题、第2遍逐句听写关键词、第3遍跟读模仿
2.2 🗣️ 口语突破 2-3周 Part 1：个人信息+日常话题20个常考问题的标准回答；Part 2：看图讨论+搭档互动（用And you?/What do you think?保持对话）。四维评分训练：语法词汇+发音+互动沟通+整体表现
2.3 📖 阅读+写作 2-3周 阅读：Part 1短通知→Part 2人物匹配→Part 3长文选择→Part 4完形→Part 5开放填空，每Part专项训练。写作：Part 6邮件（Dear开头+3要点全覆盖）→Part 7看图故事（时态一致+35词+）

🔤 模式B：综合提升（基础较好）

周次综合话题听力口语阅读写作
第7-8周🏡 日常与家庭家庭对话理解自我介绍+家人描述信息匹配题×2邮件：介绍自己
第9-10周🎒 学校与学习校园广播理解讨论学校生活选择题×3邮件：邀请朋友
第11-12周🛍️ 购物与外出商店对话+价格购物角色扮演填空题×3看图写故事×2
第13-14周✈️ 旅行与计划交通信息抓取讨论假期计划综合题型×3邮件：告知行程
⚠️ 第二阶段关键：无论A模式还是B模式，每周都要保留1小时给第一阶段学过的语法做复习。语法一旦松懈，写作和口语会明显退步。

📅 第二阶段节奏（共80节课 · 8周）

每个板块2-3周集中攻克。周一到周五每天2节。周末1套听力磨耳朵（自主完成）。

天第1节（25分钟）第2节（25分钟）
第1天 🎧 听力技巧讲解（如：数字抓取/时间识别） 🎧 真题听力实战 + 逐题讲解
第2天 🗣️ 口语话题准备（词汇+句型） 🗣️ 模拟问答 + 录音回听 + 纠正
第3天 📖 阅读题型技巧（如：信息匹配三步法） 📖 限时阅读实战 + 讲解
第4天 ✏️ 写作结构训练（邮件格式/看图写作） ✏️ 限时写作 + 对照范文修改
第5天 🔄 本周弱项回顾 + 语法快测 📊 综合小测 + 错题分析 + 下周预告
⚡ 25分钟听力课技巧：前10分钟讲技巧，中间10分钟听题+做题，最后5分钟讲错题。一篇对话反复听3遍，比泛听5篇有效得多。

📅 第三阶段节奏（共60节课 · 6周）

每周2次模考 + 3天错题复盘。最后一周轻量调整。

天第1节（25分钟）第2节（25分钟）
模考日×2 📝 限时完成阅读+写作部分 🎧 听力部分 + 即时对答案
复盘日×3 🔍 错题归类分析（四色标记法） 🔄 同类型错题重练 + 语法补漏
  • 听力：使用KET真题听力音频，每天15分钟泛听+每周1次精听（逐句听写）
  • 口语：每周围绕1个真题话题准备3-5句回答，录音自评，老师点评
  • 阅读：限时训练（每篇8-10分钟），建立"先看问题→扫读定位→精读确认"三步法
  • 写作：每周1篇邮件+1篇看图写话，对照范文修改，积累5-8个高分句型
  • 错题本：建立"四色错题本"——红色：粗心错；蓝色：知识盲区；绿色：同义替换没认出；黄色：逻辑理解错误

第三阶段：全真模考 · 查漏补缺（第15-20周）

📝 模考安排（每周2次，共12套）

周次模考重点复盘重点
第15周真题模考 ×2（不计时）熟悉全卷结构，建立答题节奏感
第16周真题模考 ×2（开始计时）阅读速度管理，听力提前读题
第17周真题模考 ×2（严格限时）易错题型归类，薄弱项标记
第18周真题模考 ×2（模拟考场）时间分配优化，心态管理
第19周模拟题 ×2（查漏补缺）只练薄弱题型，不做已掌握的
第20周考前冲刺 ×1 + 心态调整重点复习错题本，不再学新内容

🔄 错题复盘流程

  1. 标色归类：每道错题用四色系统分类（粗心/盲区/同义替换/逻辑）
  2. 补漏练习：同类型错题再找3道练习，直到连续3次正确
  3. 错题重做：每周日回看本周全部错题，遮住答案重做
  4. 考前回顾：最后一周只复习错题本中最顽固的10%
