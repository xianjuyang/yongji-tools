## 二、每日发布SOP（标准操作流程）

### 公众号发布SOP

1. **排版**（15分钟）
   - 用秀米或135编辑器打开文章
   - 粘贴文章内容
   - 设置格式：正文15px，行间距1.75，段间空行
   - 重点内容加粗或加色块
   - 配图3-5张（封面900×383px + 文中配图 + 底部二维码）

2. **预览**（5分钟）
   - 发到自己手机预览
   - 检查排版在手机上是否正常
   - 检查所有链接是否可点击

3. **发布**（2分钟）
   - 最佳时间：早上8:00-9:00 或 晚上20:00-21:00
   - 群发选择"全部用户"

4. **发布后**（10分钟）
   - 转发到自己的朋友圈
   - 转发到相关家长群（如果有的话）
   - 回复文章评论区（前几条评论1小时内回复）

### 小红书发布SOP

1. **做图**（20分钟/篇）
   - 打开Canva，选"小红书图文"模板（1080×1440）
   - 按照文案中的图片规划做5-6张图
   - 封面图最重要——标题要大字、醒目、一眼看懂
   - 统一品牌色（建议：深蓝#1a3a5c 或 品牌主色）
   - 右下角统一放井说logo小图

2. **写正文**（5分钟）
   - 复制准备好的文案
   - 检查emoji、换行在手机上是否正常
   - 正文末尾不要放微信号（会限流）

3. **加标签**（2分钟）
   - 必加：`#英语学习 #KET #英语演讲`
   - 根据内容加：`#育儿 #英语启蒙 #小学生 #英语辩论`

4. **发布**（2分钟）
   - 最佳时间：中午12:00-13:00 或 晚上20:00-22:00
   - 发布时点"标记地点"（增加本地曝光）

5. **发布后**（10分钟）
   - 自己先点赞+收藏
   - 发一条引导评论："你家孩子学英语最大的困难是什么？"
   - 检查并回复所有评论
   - 有私信立刻回复（响应速度影响账号权重）

### 抖音发布SOP

1. **剪辑**（15-20分钟/条）
   - 打开剪映
   - 导入拍摄素材
   - 自动生成字幕 → 手动校对错字
   - 加封面模板（在剪映里做封面）
   - 片尾加2秒尾帧

2. **写标题**（3分钟）
   - 标题2-3行，包含关键词
   - 加emoji增加吸引力
   - 不要放微信号（会被和谐）

3. **加标签**（1分钟）
   - `#英语学习 #育儿 #英语启蒙 #KET`

4. **发布**（2分钟）
   - 最佳时间：中午12:00-13:00 或 晚上19:00-21:00
   - 发布时选高清画质

5. **发布后**（10分钟）
   - 自己评论一条引导互动
   - 回复所有评论
   - 如果2小时内播放<500：检查标题和封面，考虑重发

### 视频号发布SOP

1. **发布**（3分钟）
   - 微信→发现→视频号→发表
   - 上传同一条视频（跟抖音一样的内容）
   - 标题稍微改一下（更正式一点）
   - 可以挂公众号文章链接

2. **发布后**（5分钟）
   - 转发到自己的朋友圈
   - 转发到微信群

## 三、内容生产节奏（每周）

```
周六或周日：
  - 写完下周3篇公众号文章
  - 准备下周7篇小红书图文素材
  - 拍摄/准备下周3条抖音视频素材

周一至周五：
  - 按照日历每天发布
  - 每天花20分钟回复各平台评论/私信

周日：
  - 复盘本周数据
  - 规划下周选题方向
```

**关键原则**：周末集中生产 → 工作日只是发布和互动。不要每天临时赶内容，质量保不住。

## 四、私信转化SOP（有人私信了怎么聊）

### 小红书/抖音私信回复模板

**用户私信**："你好，想了解一下"

**第一轮回复**（5分钟内）：
> 你好呀！👋 方便告诉我孩子的年级和目前的英语水平吗？比如有没有考过KET/PET，或者在学校英语大概什么情况？我帮你看看适合什么方案～

**用户回复后**：
> 了解了！根据你说的情况，[简单分析几句话]。详细的方案我可以微信发给你，方便加一下吗？我微信是 xxxxx

**关键原则**：
- 不要一上来就推销
- 先了解 → 再给价值 → 最后引导加微信
- 回复速度要快（最好5分钟内）
- 语气要像聊天，不像客服

## 六、常见问题预先回答

**Q：发了没人看怎么办？**
A：前两周没人看是正常的。关键是坚持发。小红书和抖音的算法需要一定量的内容来判断"你是谁、你的内容适合谁"。前20-30条是冷启动期，坚持住。

**Q：能不能花钱投流？**
A：第一个月不建议投。先积累20-30条内容，看清楚哪种内容自然数据好，再针对性地投"薯条"或"DOU+"放大。不要投没有数据验证过的内容。

**Q：要不要买粉丝？**
A：绝对不要。买的粉丝是假的，不会互动、不会转化。而且平台算法能识别——假粉丝比例高的账号会被降权。

**Q：视频不想露脸怎么办？**
A：不露脸完全可以。替代方案：
- 拍手写白板/平板（你在旁边配音）
- 做图文动画（剪映有很多模板）
- 拍学生作品/成果 + 画外音
- 做信息图动画

但说实话——露脸的内容数据普遍更好，因为用户想看到"真实的人"。

**Q：一个人做太累了怎么办？**
A：按这个节奏，第一个月你每天大概需要1-2小时（不含周末集中生产）。如果坚持不了，第二个月找兼职助手帮忙做图+剪辑——你只负责写文章和出镜拍视频。
