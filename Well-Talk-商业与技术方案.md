## 1. 品牌体系

### 1.1 命名

| | 名称 | 来源 |
|---|---|---|
| **中文名** | 永汲教育咨询 | 井卦大象传：「木上有水，井。君子以劳民劝相。」 |
| **英文名** | Well Talk | Well = 水井 + 出色地；Talk = 言说、对话 |

**「永汲」含义**：
- 永 → 恒久、深远、不竭之泉源。井之水，取之不尽，用之不竭
- 汲 → 引水、取用、从知识之源汲取学问
- 合意：**从智慧之井中汲取，以出色的表达传递给世界**

《彖传》曰：「井养而不穷也」—— 井水滋养万物，永不枯竭。教育亦当如是。

## 2. 法律合规路径

### 2.1 当前资质评估

| 资质 | 状态 | 说明 |
|---|---|---|
| 营业执照 | ✅ 已有 | 含「教育咨询服务」 |
| 互联网信息服务（经营范围） | ⚠️ 待确认 | 决定能否做 ICP 备案 |
| ICP 备案 | ⚠️ 待办 | 网站上线必需 |
| ICP 经营许可证 | ❌ 暂不需要 | 钱不进网站则无需 |
| 办学许可证 | ❌ 不需要 | 不宣称「培训学校」，用「教育咨询」 |

### 2.2 合规红线

| 🚫 不能做的事 | 为什么 |
|---|---|
| 对外宣称「XX 培训学校/机构」 | 需要办学许可证 |
| 学科类培训（语数英应试提分） | 「双减」政策严厉打击 |
| 网站上直接收学费（没有 ICP 经营许可证） | 违规经营 |
| 招聘在中国境内的外教 | 需要聘外资质（极难） |

### 2.3 合规路径

**第一步：确认/补充经营范围**

```
当前有的：教育咨询服务
需要加的：互联网信息服务（不含新闻、出版、教育等前置审批项目）
```

- 费用：自己办几十元，代办 300-800 元
- 时间：1-2 周
- 地点：当地市场监管局 / 线上国家企业信用信息公示系统

**第二步：企业 ICP 备案**

- 前提：营业执照含「互联网信息服务」+ 已购买服务器
- 费用：免费（阿里云/腾讯云协助提交）
- 时间：约 20 个工作日
- 注意：必须如实填写网站内容，不能像「坚说英语」那样挂羊头卖狗肉

**第三步：合规收款（绕过 ICP 经营许可证）**

以下方式均不需要 ICP 经营许可证：

| 渠道 | 方式 |
|---|---|
| 微信个人收款 | 加微信 → 转账/红包 |
| 小鹅通 | 小鹅通替你处理支付合规 |
| 淘宝 | 虚拟服务类目，支付宝担保交易 |
| 抖音小店/学浪 | 平台内置支付体系 |

### 2.4 案例参考：坚说英语的合规风险

| 维度 | 坚说英语实际情况 | 风险 |
|---|---|---|
| 域名 | jianshuo.tumiaoya.com（子域名） | 个人备案做商业经营 |
| ICP 备案 | 京ICP备14061881号（个人备案） | 备案内容为「图片压缩」 |
| 实际业务 | 在线英语约课、课包收费 | 与备案内容完全不符 |
| 合规状态 | ⚠️ 严重不合规 | 随时可能被举报关停 |

> **教训**：不要学他们。走正规企业备案路径。

## 3. 商业模式设计

### 3.1 平台模式

```
                          Well Talk 永汲教育
                               │
              ┌────────────────┼────────────────┐
              │                │                │
         外教入驻          学生预约          平台抽佣
      （30-40 人）      （选老师+时间）     （20-30%）
              │                │                │
              └────────────────┴────────────────┘
                               │
                         佣金进入平台账户
                               │
                    第三方代发 → 外教收到工资
```

### 3.2 资金流向

```
学生付 ¥150/节
      ↓
平台收款（小鹅通/淘宝/抖音/微信）
      ↓
平台抽佣 20-30%（¥30-45）
      ↓
外教到手 ¥105-120（通过 Payoneer/Wise/连连支付代发）
```

### 3.3 定价参考

| 课程类型 | 建议定价（每 25 分钟） | 外教分成 | 平台佣金 |
|---|---|---|---|
| KET/PET 口语 | ¥120-180 | ¥90-130 | ¥30-50 |
| 演讲训练 | ¥180-280 | ¥130-200 | ¥50-80 |
| 辩论训练 | ¥200-300 | ¥140-210 | ¥60-90 |
| 雅思/托福口语 | ¥150-250 | ¥105-175 | ¥45-75 |
| 综合英语/自由对话 | ¥100-150 | ¥75-110 | ¥25-40 |

### 3.4 竞争壁垒

```
1. 品控体系     → 外教审核 + 试讲 + 课后评价
2. 教材/内容    → 自研课程体系，外教按平台教案上课
3. 课后服务     → 作业批改 + 学习报告（外教自己做不了）
4. 封闭通讯     → 学生和外教不在平台外联系
5. 预付课包     → 学生预付给平台 → 平台按课时结算给外教
```

## 5. SaaS 平台调研对比

### 5.1 价格与功能总览

| 平台 | 入门版 | 主力版 | 旗舰版 | 交易抽成 | 约课支持 |
|---|---|---|---|---|---|
| **小鹅通** | ¥6,800/年 | ¥12,800/年 | ¥25,800/年 | 超出免额 2%+0.6% | ❌ 不支持 |
| **有赞教育** | ¥3,800/年 | ¥6,800/年 | ¥12,800/年 | 封顶 ¥0.50/单+0.6% | ⚠️ 基础 |
| **短书** | ¥3,888/年 | ¥6,988/年 | ¥12,888/年 | 阶梯费率 | ❌ 不支持 |
| **ClassIn** | 免费试用 | 预充值扣费 | 企业定制 | 无抽成 | ❌ 不支持 |

### 5.2 专业教务平台（支持约课）

| 平台 | 约课能力 | 小程序 | 价格 | 亮点 | 缺点 |
|---|---|---|---|---|---|
| **虎鲸爱学** | ⭐⭐⭐⭐⭐ | ✅ | 免费基础版 | 智能排课+消课+AI反馈 | 微官网展示能力弱 |
| **爱耕云** | ⭐⭐⭐⭐⭐ | ✅ | 弹性计费 | 约课软著持有者 | 品牌感一般 |
| **橙子约课通** | ⭐⭐⭐⭐ | ✅ | ¥199/人/年起 | 轻量低价 | 营销工具弱 |
| **满班** | ⭐⭐⭐⭐ | ✅ | 增值付费 | 可视化课表 | 界面较传统 |

### 5.3 关键结论

> **小鹅通 = 好看的前台 + 弱的教务（不支持约课）**  
> **虎鲸爱学们 = 强的教务 + 丑的前台（不支持营销）**  
> **你需要的是：小鹅通做合规外壳 + 自嵌入约课系统 = 两者兼得**

## 6. 约课系统方案

### 6.1 三种解决方案

| 方案 | 说明 | 成本 | 灵活性 |
|---|---|---|---|
| **A: 纯手动** | 微信排课，聊天记录管理 | ¥0 | ⭐ |
| **B: 教务 SaaS 嵌入** | 虎鲸爱学等嵌入小鹅通 | ¥0-几千/年 | ⭐⭐⭐ |
| **C: 自建约课系统** | AI 辅助开发，部署到服务器 | ~¥900/年（服务器） | ⭐⭐⭐⭐⭐ |

### 6.2 自建约课系统功能规划

**外教端**：
- 设置每周固定档期（哪天、几点到几点）
- 临时关闭某个时段
- 查看自己的预约列表
- 查看学生信息
- 收入统计

**学生端**（公开页面）：
- 浏览外教列表
- 选课程类型
- 日历视图选日期
- 查看可用时段（25 分钟/节 + 5 分钟休息，支持连堂）
- 填写信息提交预约

**管理端**（你的后台）：
- 查看/审核所有预约
- 按状态筛选（待确认/已确认/已取消）
- 学生档案（点击姓名 → 全部历史记录）
- 月度汇总导出 Excel
- 佣金自动计算

### 6.3 时间表规则

```
课程单元：25 分钟/节
课间休息：5 分钟
全天时段：08:00 - 21:30（共 28 个可选时段）
连堂支持：1节(25min) / 2节(50min) / 3节(75min) / 4节(100min)
休息日：周日
```

### 7.2 ICP 备案流程

```
① 确认/补充营业执照经营范围（含「互联网信息服务」）
② 阿里云购买域名 + 轻量服务器
③ 在阿里云提交企业 ICP 备案申请
④ 阿里云初审（1-2 天）
⑤ 工信部审核（约 20 个工作日）
⑥ 备案通过 → 域名解析 → 网站上线
```

| 维护事项 | 频率 | 说明 |
|---|---|---|

| 数据备份 | 每月 | 系统自带 Excel 导出，一键备份 |
| SSL 证书 | 自动续期 | 阿里云免费，无需操作 |
| 功能更新 | 按需 | 修改 HTML 文件上传即可 |

### 7.4 数据安全升级路径
